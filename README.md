# DexPlayer

A modern video player for Android and Samsung DeX, built on top of [mpv-android](https://github.com/mpv-android/mpv-android).

## About

DexPlayer is a fork/client of mpv-android with a redesigned UI using Material You (Material 3) and optimized for Samsung DeX desktop mode.

**Core engine:** [MPV](https://mpv.io/) via mpv-android  
**UI:** Jetpack Compose + Material 3  
**Target:** Android 15 (API 35), Samsung DeX
